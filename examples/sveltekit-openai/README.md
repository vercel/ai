# `sveltekit-openai`

To run this example, clone the repository, and at the root install dependencies and build:

```
pnpm install
pnpm build
```

Then create a `.env.local` file in `examples/sveltekit-openai` with:

```
OPENAI_API_KEY=<your key>
```

Then, run:

```bash
pnpm -F sveltekit-openai dev
```
