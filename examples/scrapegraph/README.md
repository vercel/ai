# ScrapeGraph AI SDK Examples

This directory contains examples demonstrating the usage of the ScrapeGraph AI provider with the Vercel AI SDK.

## Setup

1. Set your ScrapeGraph AI API key:

```bash
export SCRAPEGRAPH_API_KEY=your-api-key
```

Or create a `.env` file:

```
SCRAPEGRAPH_API_KEY=your-api-key
```

2. Install dependencies:

```bash
pnpm install
```

3. Run the examples:

```bash
pnpm start
```

## Examples Included

- **SmartScraper**: Extract structured data from websites
- **SearchScraper**: Search and extract data from multiple sources
- **Markdownify**: Convert webpages to markdown
- **Scrape**: Get raw HTML content
- **Crawl**: Multi-page crawling with AI extraction
- **Agentic Scraper**: Complex workflows with AI agents
- **Sitemap**: Extract website structure
