# AI_InvalidToolContextError

This error occurs when a tool's `toolsContext` entry does not match the tool's `contextSchema`.

## Properties

- `toolName`: The name of the tool with invalid context
- `toolContext`: The invalid tool context value
- `message`: The error message
- `cause`: The underlying validation error

You can check if an error is an instance of `AI_InvalidToolContextError` using:

```typescript
import { InvalidToolContextError } from 'ai';

  // Handle the error
}
```
