# `LanguageModelMiddleware`

`LanguageModelMiddleware` is the stable, user-facing middleware type exported from `ai`.
It enhances language model behavior by intercepting and modifying calls to the model.
You can use it to add features like guardrails, RAG, caching, and logging in a language model agnostic way.

See [Language Model Middleware](/docs/ai-sdk-core/middleware) for more information.
