# `generateTranscript()`

<Note type="warning">`generateTranscript` is an experimental feature.</Note>

Generates a transcript from an audio file.

It is ideal for use cases where you need to generate a transcript from an audio file,
such as creating a transcript for a podcast or a video.

```ts
import { experimental_generateTranscript as generateTranscript } from 'ai';

const { transcript } = await generateTranscript({
  model: openai.transcription('whisper-1'),
  audio: audioData,
});

console.log(transcript);
```
