# Doubleword Provider

[Doubleword](https://doubleword.ai) is an AI model gateway providing unified routing, management, and security for inference across multiple model providers. The Doubleword provider for the AI SDK lets you access all models configured in your Doubleword control layer through a single integration, with up to 90% cost savings via the [Doubleword Inference API](https://docs.doubleword.ai) batch pricing.

## Setup

The Doubleword provider is available in the `@doubleword/vercel-ai` module. You can install it with:

## Provider Instance

You can import the default provider instance `doubleword` from `@doubleword/vercel-ai`:

```ts
import { doubleword } from '@doubleword/vercel-ai';
```

If you need a customized setup, you can import `createDoubleword` and create a provider instance with your settings:

```ts
import { createDoubleword } from '@doubleword/vercel-ai';

const doubleword = createDoubleword({
  apiKey: process.env.DOUBLEWORD_API_KEY,
});
```

The provider uses the `DOUBLEWORD_API_KEY` environment variable by default. You can also authenticate via `~/.dw/credentials.toml`.

## Language Models

You can create language models using any model ID configured in your Doubleword control layer:

```ts
import { doubleword } from '@doubleword/vercel-ai';

const model = doubleword('Qwen/Qwen3.5-35B-A3B-FP8');
```

Doubleword routes requests to the appropriate inference provider based on your configured routing rules.

## Embedding Models

You can create embedding models using the `.textEmbeddingModel()` factory method:

```ts
import { doubleword } from '@doubleword/vercel-ai';

const embeddingModel = doubleword.textEmbeddingModel('Qwen/Qwen3-Embedding-8B');
```

## Examples

### `generateText`

```ts
import { generateText } from 'ai';
import { doubleword } from '@doubleword/vercel-ai';

const { text } = await generateText({
  model: doubleword('Qwen/Qwen3.5-35B-A3B-FP8'),
  prompt: 'What is Doubleword?',
});

console.log(text);
```

### `streamText`

```ts
import { streamText } from 'ai';
import { doubleword } from '@doubleword/vercel-ai';

const result = streamText({
  model: doubleword('Qwen/Qwen3.5-35B-A3B-FP8'),
  prompt: 'Explain the benefits of an AI gateway.',
});

for await (const chunk of result.textStream) {
  process.stdout.write(chunk);
}
```

### Tool Calling

```ts
import { generateText, tool, jsonSchema, stepCountIs } from 'ai';
import { doubleword } from '@doubleword/vercel-ai';

const { text } = await generateText({
  model: doubleword('Qwen/Qwen3.5-35B-A3B-FP8'),
  tools: {
    getWeather: tool({
      description: 'Get the weather for a location',
      inputSchema: jsonSchema({
        type: 'object',
        properties: {
          location: { type: 'string', description: 'The city name' },
        },
        required: ['location'],
        additionalProperties: false,
      }),
      execute: async ({ location }) => `Sunny, 22°C in ${location}`,
    }),
  },
  stopWhen: stepCountIs(5),
  prompt: 'What is the weather in London?',
});
```

### Embeddings

```ts
import { embed } from 'ai';
import { doubleword } from '@doubleword/vercel-ai';

const { embedding } = await embed({
  model: doubleword.textEmbeddingModel('Qwen/Qwen3-Embedding-8B'),
  value: 'sunny day at the beach',
});
```

### Batch Pricing

For background workloads where latency is not critical, use `createDoublewordBatch` to transparently route requests through the [Doubleword Inference API](https://docs.doubleword.ai) Batch API — cutting inference costs by up to 90%. Powered by [`autobatcher`](https://www.npmjs.com/package/autobatcher) under the hood.

```ts
import { generateText } from 'ai';
import { createDoublewordBatch } from '@doubleword/vercel-ai';

const doubleword = createDoublewordBatch();

// Concurrent calls are automatically collected into batch submissions
const results = await Promise.all([
  generateText({ model: doubleword('Qwen/Qwen3.5-35B-A3B-FP8'), prompt: 'Summarize chapter 1.' }),
  generateText({ model: doubleword('Qwen/Qwen3.5-35B-A3B-FP8'), prompt: 'Summarize chapter 2.' }),
  generateText({ model: doubleword('Qwen/Qwen3.5-35B-A3B-FP8'), prompt: 'Summarize chapter 3.' }),
]);

// Flush remaining requests when done
await doubleword.close();
```

The interface is identical to the real-time provider — only streaming is not supported (batch results return all at once).
